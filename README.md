# 프로젝트 소개
- 한 줄 소개: 펫시터 매칭 서비스로 펫시터를 매칭하여 예약하고 리뷰를 남길 수 있다.
- 배경: 펫시터 매칭 서비스를 만들면서 API서버를 만드는 것과 웹페이지가 Axios를 통해 API서버와 상호작용하게 만드는 법을 배우고자 함.
- 참여인원: 4명
- 본인의 역할: API작성 및 에러 핸들링, 프론트에서 axios를 이용한 API서버와의 통신.
- 진행기간: 7일
